# AGENTS.md

## UI / TUI Direction

ROOM should lean into strange modular-synth outsider data art, not generic dashboard polish.

- Preserve and extend the existing neon control-surface feel.
- Favor instrument-panel metaphors over business-app metaphors: oscillators, orbits, traces, scopes, meters, signal paths, lock indicators, drift, saturation, phase, resonance.
- Prefer visuals that feel like experimental operator tooling or electronic art objects, as long as they still communicate state clearly.
- Use motion with purpose. Animation should suggest changing system state, not just decorate the screen.
- Abstract visualizations are good when they map to something real in the run: progress, failures, pivots, retries, momentum, queue depth, or event activity.
- Avoid bland enterprise charts, generic cards, and default loading widgets unless the task explicitly calls for them.
- Keep text legible and contrast high. Weird is good; unreadable is not.
- When inventing new panels or graphs, bias toward surprising but disciplined shapes and labels rather than standard admin UI patterns.
